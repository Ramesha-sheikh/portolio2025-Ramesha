This directory contains Claude Code CLI configuration and memory for this project.

## Structure

```
.claude/
├── settings.json          # Project-specific Claude settings
├── memory/               # Project memory (auto-managed by Claude)
│   └── MEMORY.md        # Main memory file
└── README.md            # This file
```

## Files

### settings.json
Project configuration including:
- Project metadata (name, version, description)
- Developer preferences
- File path shortcuts
- Common commands
- Tech stack information

### memory/
Directory for Claude to store project-specific learnings and patterns.
Auto-managed by Claude Code CLI.
